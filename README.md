# Prova Técnica – Nasajon

Script em Python que processa uma lista de municípios a partir de um arquivo CSV,
consulta dados oficiais na API do IBGE e gera um arquivo `resultado.csv`
com os dados enriquecidos e o status de cada município.

O resultado.csv é gerado automaticamente a partir do código, conforme as regras
definidas no enunciado da prova.

Ao final da execução, o programa também calcula e exibe no terminal
as estatísticas solicitadas (totais e médias por região).
